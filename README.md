# Coffe-Machine-simulator-with-Java

## Overview
Console coffee machine simulator implementing OOP concepts: resources, menu actions, state, and money box. Hyperskill project.

## Tech
- Java 17+

## How to run

```bash
javac -d out src/*.java
java -cp out Main
```

## Features
- Drinks: espresso / cappuccino / latte
- Actions: buy, fill, take, remaining, exit
- Resource checks & messages
